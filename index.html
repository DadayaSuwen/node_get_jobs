🍀 Node.js Boss直聘自动投递工具

📖 项目简介

基于Node.js和Puppeteer开发的Boss直聘自动投递工具，支持AI智能筛选和自动打招呼功能。

🚀 功能特点

  • ✅ 自动登录Boss直聘（扫码登录）
  • ✅ 智能岗位搜索和筛选
  • ✅ AI智能打招呼（DeepSeek集成）
  • ✅ 配置文件驱动，易于定制
  • ✅ 自动化Cookie管理
  • ✅ 详细的投递日志记录
  • ✅ 黑名单过滤系统

⚙️ 快速开始

1. 克隆项目

git clone https://github.com/DadayaSuwen/node_get_jobs.git
cd node_get_jobs

2. 安装依赖

npm install

3. 配置设置

编辑 config.json 文件：

{
  "sayHi": "您的打招呼语",
  "keywords": ["前端", "React", "TypeScript"],
  "cityCode": ["101280100"],
  "industry": ["100001"],
  "experience": ["102"],
  "enableAI": true,
  "api": {
    "baseUrl": "https://api.deepseek.com/chat/completions",
    "apiKey": "your-api-key",
    "model": "deepseek-chat"
  }
}

4. 运行程序

npm start

📁 配置文件说明

  • config.json - 主配置文件（投递参数、AI设置）
  • config_test.json - 测试环境配置
  • data.json - 黑名单和过滤规则
  • cookie.json - 自动生成的登录凭证
